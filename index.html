Data Visualization Project

This data visualization project presents an overview on Daily Avg Precipiration, Daily Avg Min Temperature & Daily average Max Temperature for all US States.

Click on the below button to go back to Data Overview page which presents an overview of precipitation,Min & Max temperatures for all 50 US states

From the overview page click on drill down to navigate to webpage to view historic line chart providing data per state for over 80 years.

The project uses clustered bar chart to provide an overview data of weather parameters for all 50 states

While it uses multi series line charts to provide information on how the measures changed over time per state
